This folder contains the source code of the DecorNest website developed for the Web Technology Innovative Assignment.

The website is built using HTML, CSS, JavaScript, jQuery, and Bootstrap. It provides customized home decor ideas including categories like living room decor, bedroom decor, DIY ideas, and product pages.

Files included:
index.html – Main homepage of the website
style.css – Styling and layout design
script.js – JavaScript functionality
images – Images used in the website
